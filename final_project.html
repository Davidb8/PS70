Final Project

For my final project, I have chosen to explore the field of signal processing in the context of communications between space machinery. Specifically, I want to look into communication between a base station and rover. I envision a rover that is able to record some data like imagery or temperature. It then takes that data and returns it back to the base stations using short range communication, and then the base station uses a longer range form of communication to transmit the data off world.
